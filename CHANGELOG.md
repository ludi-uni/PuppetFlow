# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Added

- Headless **CLI** (`pf run`) with YAML config (`puppetflow.yaml`)
- `@puppetflow/cli-config`, `@puppetflow/runtime-launcher` packages
- Studio **CLI config export** with save-location picker
- Studio settings persistence (`localStorage`)
- PFScript language core, Studio PFScript tab, and official samples
- Standard preset model (PFScript + blink/idle + graph)

### Changed

- Studio hooks and `features/{shared,simple,expert}` layout
- Documentation sync for architecture, roadmap, and guides
