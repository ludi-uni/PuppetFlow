# PuppetFlow 追加仕様

# Phase 1: Micro Behavior対応

## 目的

PuppetFlowへ軽量なMicro Behavior機能を追加する。

目的はショート動画向けの派手な演出ではなく、

- 存在感向上
- 思考感向上
- 観測感向上
- 待機中の自然さ向上

である。

本フェーズではアニメーションレイヤーシステムは導入しない。

Behavior名を受け取り、対応するVMCタイムラインを実行する最小構成とする。

---

# システム構成

```text
Behavior Engine
        ↓
Micro Behavior Engine
        ↓
PuppetFlow
        ↓
VMC
```

---

# 実装範囲

実装対象

```text
Behavior Registry
Behavior Executor
Behavior Queue
```

実装しないもの

```text
Animation Layer
Blend Tree
Motion Graph
Behavior Learning
```

---

# Behavior Registry

目的

Behavior名と実際のVMC動作を紐付ける。

---

## 定義例

```yaml
look_up:
  duration: 1.2

head_tilt:
  duration: 1.5

small_nod:
  duration: 0.8

long_blink:
  duration: 0.5

look_left:
  duration: 1.0

look_right:
  duration: 1.0
```

---

# Behavior API

Micro Behavior Engineから受信する形式

```json
{
  "behavior": "look_up"
}
```

---

将来拡張対応

```json
{
  "behavior": "look_up",
  "strength": 0.8
}
```

# パラメータ補間

線形補間は禁止。

必ずEase In / Ease Outを使用する。

---

例

```text
Bad

0.0
↓
1.0
```

---

```text
Good

0.0
↓
0.3
↓
0.7
↓
1.0
```

---

目的

ロボット感の軽減。

# Behavior Queue

目的

Behavior衝突防止。

---

仕様

同時実行数

```yaml
maxConcurrent: 1
```

---

新規Behavior受信時

```text
実行中
↓
Queueへ追加
```

---

FIFO方式。

---

# Cooldown

同じBehaviorの連続実行防止。

---

例

```yaml
look_up:
  cooldown: 5s

head_tilt:
  cooldown: 8s

small_nod:
  cooldown: 3s

long_blink:
  cooldown: 2s
```

---

Cooldown中は無視する。

---

# ランダム化

同じBehaviorでも微小変動を加える。

---

例

```yaml
look_up:

eyeY: 0.35～0.45

duration: 1.0～1.4
```

---

目的

パターン化防止。

---
