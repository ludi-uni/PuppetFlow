# Official presets

Canonical `.pfpreset` files for Studio and Playground.

- **Edit via:** `pnpm build:presets` (`packages/preset/src/build-official-presets.ts`)
- **Mirror:** repo root `presets/` (kept in sync by CI)
- **Baseline:** `Standard.pfpreset` — PFScript (body/breath) + Graph (mouthX) + blink/idle

Do not hand-edit generated JSON without updating the build script.
